# オセロダッシュボード

学生が作成したオセロゲームの Python コードを、ブラウザ上で Pyodide を使って実行・プレイできる Web アプリケーションです。

## データ構造

### JSON ファイル形式

各データファイル（`10_3.json`, `10_4.json`, `11_3.json`, `11_4.json`）は、以下の構造を持つオブジェクトの配列です：

```json
[
  {
    "name": "Shimon",
    "description": "東大王のクイズ×オセロを自分の好きな日本史に置き換えて作りました。",
    "code": "# ========================================\n# 日本史クイズオセロ\n..."
  },
  {
    "name": "Yugo",
    "description": "本プログラムは、Google Colab 上で動作する...",
    "code": "import random\n\nclass OthelloBoard:\n..."
  }
]
```

**フィールド説明:**

- `name` (string): 学生のファーストネーム
- `description` (string): コードの説明・工夫した点（10*系のみ、11*系は空文字列）
- `code` (string): 提出された Python コード（`_x000D_`キャリッジリターンは削除済み）

## ライセンス

教育目的のプロジェクトです。
